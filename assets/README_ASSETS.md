# Asset Pipeline

Recommended free pipeline:
- Blender: modeling, rigging, animation, UV
- Krita: concepts, UI paintings, texture touch-up
- Material Maker: procedural PBR materials
- Poly Haven: CC0 HDRIs/textures/models; still keep an asset license manifest

Create these folders later:
- characters/
- outfits/
- environment/
- props/
- animations/
- materials/
- audio/
- ui/
