# Art Direction

## Visual Target
“Photorealistic” ထက် “optimized cinematic realism” ကိုဦးစားပေးပါ။
- PBR materials
- Realistic fabric folds and metal/wood response
- Warm gold/red/indigo palette
- Cinematic dusk/night lighting
- Medium-detail environment geometry
- LODs and baked lighting where possible

## Character Design Order
1. Silhouette sheet
2. Social role/personality
3. Outfit layers and color variants
4. Front/side/back turnaround
5. Low-poly game model
6. UV + PBR textures
7. Rig
8. Walk/run/jump/idle/emotes
9. Import test in Godot
10. Multiplayer performance test

## Cultural Safety
- Fictional kingdom names and symbols
- Museum references and historical research before final outfits
- Sacred imagery ကို casual decoration အဖြစ်မသုံးရန်
- Dynasty တစ်ခု၏ ဝတ်စုံများကို အခြားခေတ်နှင့် မရောမီ reference sheet ထုတ်ရန်
