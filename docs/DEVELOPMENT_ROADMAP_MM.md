# Development Roadmap

## Milestone 1 — Foundation
- Third-person controller
- Offline map
- Host/Join multiplayer
- Placeholder character

## Milestone 2 — Vertical Slice
- One polished courtyard/village
- One final-quality hero character
- Basic animations
- One co-op quest
- Interaction prompts and UI

## Milestone 3 — Social Systems
- Lobby and room code
- Emotes
- Text chat with report/mute tools
- Player names
- Cosmetics

## Milestone 4 — Persistence
- Account/authentication
- Inventory and progression database
- Server validation
- Reconnect handling

## Milestone 5 — Optimization and Release
- LODs
- Occlusion/culling
- Texture compression
- Network bandwidth profiling
- Desktop build first
- Android build after stable desktop prototype
