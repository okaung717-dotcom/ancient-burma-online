# Game Design Blueprint — “THE GOLDEN REALM”

## High Concept
Ancient Myanmar ကို တိုက်ရိုက်ကူးယူခြင်းမဟုတ်ဘဲ၊ ပျူ–ပုဂံ–ကုန်းဘောင်ခေတ်တို့၏ ရိုးရာဗိသုကာ၊ အထည်အလိပ်၊ ပွဲတော်နှင့် ဒဏ္ဍာရီဆန်မှုများမှ အနုပညာလှုံ့ဆော်မှုယူထားသော fictional online social-adventure game။

## Player Fantasy
ကစားသမားသည် ရွှေနန်းပြည်၏ ခရီးသွား၊ လက်မှုပညာရှင်၊ စစ်သည်၊ ကုန်သည် သို့မဟုတ် နတ်သမိုင်းရှာဖွေသူအဖြစ် မိတ်ဆွေများနှင့် ရွာများ၊ ဘုရားပုထိုးဟောင်းများ၊ တောအုပ်နှင့် မြစ်ကြောင်းများကို လေ့လာမည်။

## First Playable Scope
- Players: 1–12
- Map: နန်းတော်ပတ်ဝန်းကျင် + ရွာသေးတစ်ရွာ
- Session: 15–30 မိနစ်
- Activities: lantern lighting, relic search, boat race prototype, cooperative gate puzzle
- Progression: cosmetic titles and outfit pieces only
- No pay-to-win

## Core Loop
1. Social hub တွင် မိတ်ဆွေတွေ့
2. Quest / activity ရွေး
3. 2–4 ယောက်ပူးပေါင်းပြီး စူးစမ်း၊ puzzle ဖြေ၊ resource စု
4. Village ပြန်လာပြီး cosmetic / decoration unlock
5. Festival event တွင် ပြသ၊ ဓာတ်ပုံရိုက်၊ mini-game ဆော့

## MVP Features
- Lobby/Host/Join
- Player movement + camera
- 1 cooperative quest
- 1 social emote wheel
- 1 collectible system
- 3 basic outfits
- Simple text chat (later moderation required)
- Save data locally first; cloud account later
